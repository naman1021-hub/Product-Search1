# Product Search and Category Filter

A React application that displays a list of products with search and category filter functionality.

## Features

- Display products with name, category, and price
- Search products by name (case insensitive, real-time)
- Filter products by category (All, Electronics, Clothing, Home)
- Combine search and filter

## Setup

1. Install dependencies:
   ```
   npm install
   ```

2. Start the development server:
   ```
   npm run dev
   ```

3. Open your browser to `http://localhost:5173` (or the port shown in the terminal).

## Usage

- Type in the search box to filter products by name.
- Select a category from the dropdown to filter by category.
- The product list updates dynamically.

## Troubleshooting

- If the app doesn't start, ensure Node.js and npm are installed.
- If there are dependency issues, try deleting `node_modules` and running `npm install` again.
- For build issues, check that all files are in the correct locations.
